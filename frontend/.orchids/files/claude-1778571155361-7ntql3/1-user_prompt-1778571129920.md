Create a modern, fully responsive website for a creative marketing agency named **Beion Marketing** using **React.js** and **Tailwind CSS**. The website should be designed with scalability in mind, using structured **JSON-based data architecture** so APIs can easily be integrated later without changing the UI components.

The agency provides services for:

* Event Videography
* Food Photography & Videography
* Corporate Video Production
* Web Development

The website should have a premium, aesthetic, and professional appearance with smooth animations, hover effects, modern layouts, and a consistent visual identity across all pages.

---

# Project Requirements

## Tech Stack

* React.js
* Tailwind CSS
* React Router DOM
* Framer Motion
* JSON-based dynamic content structure
* Reusable and scalable component architecture

# Portfolio Detail Page

Each portfolio page should dynamically load data from JSON.

Display:

* Category title
* Short description
* Related projects/videos only

Each project item must contain:

* Video title
* Company/client name
* Embedded YouTube video
* Optional project description

### Example JSON Structure

```js
const portfolioData = [
  {
    category: "food",
    title: "Food Videography",
    description: "Creative food commercials and restaurant branding videos.",
    projects: [
      {
        id: 1,
        videoTitle: "Italian Pizza Commercial",
        company: "Pizza Hub",
        youtubeUrl: "https://www.youtube.com/embed/xxxxx"
      }
    ]
  }
]
```

### Requirements

* Responsive video grid
* Clean modern cards
* Smooth transitions and hover effects
* Dynamic rendering only from JSON

# Architecture Requirements

## Important

Structure the project so all content is managed through JSON files or objects for future API integration.

Examples:

* `services.json`
* `portfolio.json`
* `socialLinks.json`
* `navigation.json`

UI components should map through JSON data dynamically instead of hardcoded content.
